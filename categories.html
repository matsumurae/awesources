How to use

  1. First, we need to know which type of resource we need.
  2. Then, search in the TOC the category. Clic on it and it will take to that section.
  3. You can view which subcategories have. I put subcategories for best organization and search.
  4. Select your subcategory, they will take you to new page with the content listed.
  5. You will see an index, where you can view which resources have this section. You can clic on it and it will take to the selected resource.

This pages contains photo of the resource, name, description, type of license and the url.
