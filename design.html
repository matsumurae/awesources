Color picker

Colorrrs

Colorrrs is a clean and simple randomizer / converter color app. The app is based on the beautiful work of Jamie Brittain (@jamiebrittain).

License: N/A

http://colorrrs.com/

Flat UI Colors

This small web app helps you to copy the colors from Flat UI. It's simple to use, but not have much colors to choose.

License: N/A

http://flatuicolors.com/

Flat UI Color Picker

FlatUIcolorpicker is a small project with FlatUI colors. Only has 6 colours, but is a good option for Flat colours.

License: N/A

http://www.flatuicolorpicker.com/

Hailpixel

Minimal website where you can select infinite (or nearly) colours, just move your mouse and clic for select the colour.

License: N/A

http://color.hailpixel.com/

Bootflat color picker

Another website for select Flat UI colours.

License: N/A

https://bootflat.github.io/color-picker.html

Palettes

Material Palette

Material Design Palettes. Generate a palette for you only picking two colours, and you can see how looks in Android!

License: N/A

http://www.materialpalette.com/

Coolors

Super fast and easy to use colour schemes generator. Have option to search for palettes created by other users.

License: N/A

http://coolors.co/

Palettab

A google chrome extension to discover nice color palletes (via colourlovers) and fonts (via google fonts).

License: MIT

http://palettab.com/

Color palette pocket

A Google Chrome extension to discover nice color palletes (via colourlovers) and fonts (via google fonts).

License: N/A

https://chrome.google.com/webstore/detail/color-palette-pocket/cpeodkcejoabdmfneaoihkjkibgeifci

Palette Ninja

Palette Ninja is an online tool that allows you to create harmonious color schemes. Have Google Chrome extension too.

License: N/A

https://chrome.google.com/webstore/detail/color-palette-pocket/cpeodkcejoabdmfneaoihkjkibgeifci

PLTTS

Easily find you matching color palette. Made by Dominik Schmidt (@iDuuck)

License: N/A

http://pltts.me/

COLOR ME CURIOUS

Compilation of palettes made by users in Tumblr. In this blog, are organized by tags.

License: None, however credit is nice.

http://www.colourpod.com/

colllor

Colllor is a web utility that converts a color from a wide range of colors thanks to which we find the one we want.

License: N/A

http://colllor.com/

0to255

It's a simple tool that helps web designers find lighter and darker colors based on any color. Have option for random colour.

License: MIT

http://www.0to255.com/

colourco.de

Pick colours based on hsl. Scheme-mode with monochromatic, analogic, complement, triad and quad colours.

License: MIT

http://www.colourco.de/

Design Seeds

Design Seeds celebrate colors found in nature and the aesthetic of purposeful living. Makes palettes from photos of nature.

License: None, however credit is nice.

http://design-seeds.com/

Patterns

In my opinion, this is the best patterns website. We can search by tags and have PS plugin.

License: CC BY-SA 3.0

http://subtlepatterns.com/

CSS3 Patterns Gallery

This is a substitute for images. We use only CSS for do patterns, not have much repertoire.

License: N/A

http://lea.verou.me/css3patterns/

SubtlePatterns Bookmarklet

Preview hundreds of great patterns from Subtle Patterns directly on your website.

License: CC Attribution-ShareAlike 3.0

http://bradjasper.com/subtle-patterns-bookmarklet/
