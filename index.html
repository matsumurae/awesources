Welcome to Awesources

Welcome to awesources! Here you'll find awesome and wonderful resources for every work you'll do. For developers, designers and inspiration-searchers.

Issues and sugestions

If you have any suggestions on how to make this better, open an issue. I appreciate your help :)

New resource

If you have a resource to contribute, make a pull request. You must send with this template:

  • Domain name
  • License
  • Description
  • Category *

You'll find categories list in categories page.

Be sure to send license-free resources. I'm in favor of free code.
